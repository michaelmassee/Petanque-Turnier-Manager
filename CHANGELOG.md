# Changelog

## Version 5.0.0

### Aktualisierungen
- **Java 25** – Mindestanforderung auf Java 25 (LTS) erhöht.
- **LibreOffice 7** – Plugin an die moderne LibreOffice-Architektur angepasst. Mindestanforderung ist nun LibreOffice 7.

### Technische Änderungen
- UNO-Komponentenregistrierung modernisiert.
- Build-System auf Gradle umgestellt.
- OXT-Dateiname wird automatisch aus `description.xml` gelesen.
